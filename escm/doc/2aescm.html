NAME

2aescm - convert an escm file into the XML format

SYNOPSIS

2aescm [-p PATH] [-o OUTFILE] INFILE

DESCRIPTION

2aescm converts the original escm format with <? and !> into the XML instruction format with <scm and ?>. #? syntax, used to specify the backend interpreter, is translated by an extended meta-argument functionaly.

OPTIONS

-p PATH
Specify the path of aescm. The default is the one installed with this program.

-o OUTFILE
Specify the output file. Without this option, the output is sent to stdout.
